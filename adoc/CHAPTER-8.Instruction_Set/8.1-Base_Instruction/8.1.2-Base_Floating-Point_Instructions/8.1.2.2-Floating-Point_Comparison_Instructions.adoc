*Usage :* 
[source]
----
fcmp.slt.s   $fcc0, $f23, $f24
bceqz        $fcc0, .L128
nop
.L128:
nop
----

* *Explanation :*

** *`fcmp.slt.s`*
*** *if* (*`$f23`* < *`$f24`*) *`$fcc0`* = *1*
*** *else* *`$fcc0`* = *0*
** *`bceqz`*
*** *if* (*`$fcc0`* == *0*) *jump* *`.L128`*
