=== *About This Book*

[.text-justify]
This book provides tutorial and reference information on the *`LoongArch`* Assembly Language Program. It covers *`LoongArch`* data types, registers, addressing, and inline assembly in C and C++ compilers. Additionally, it describes the assembly language mnemonics, pseudo-instructions, macros, and directives available to assembly language programmers. It also details the format of the *`LoongArch`* ELF file.
